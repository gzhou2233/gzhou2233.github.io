# Ma2主要知识点

## 创建show文件

Backup → New show → 清除上一个秀的设置（一般只清除前四个设置，网络设置一般别人设置好了直接用就行）

![配接灯具](https://s2.loli.net/2022/04/20/57xAgjmc6I1KruG.jpg)

**配接灯具**

Tips：拿到一个台子后保存一个空窗口（后面创建其他功能窗口有用）

**准备工作**

- 创建好一个空窗口后，现在需要一个电脑灯的列表窗口，点击空窗口的空白处
